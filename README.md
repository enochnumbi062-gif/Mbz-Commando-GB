# 🎖️ Mbz Commando GB

**Mbz Commando GB** est un jeu d'action "Run and Gun" au style Neo-Retro, inspiré par l'esthétique classique de la Game Boy originale. Développé avec **Python (Flask)** pour le backend et **JavaScript (Canvas)** pour le moteur de jeu.



## 🚀 Caractéristiques
* **Look & Feel Game Boy :** Palette de couleurs authentique (#9bbc0f, #8bac0f, #306230, #0f380f).
* **Moteur Audio 8-bit :** Sons générés en temps réel via Web Audio API (pas de fichiers MP3 lourds).
* **Gameplay Dynamique :** Effet de recul (Kickback) lors des tirs et IA ennemie réactive.
* **Zéro Installation :** Jouable directement dans le navigateur.

## 🕹️ Commandes
| Action | Touche |
| :--- | :--- |
| **Mouvement** | `Z` `Q` `S` `D` ou Flèches |
| **Tirer** | `Clic Gauche Souris` |
| **Rejouer** | `Clic` après un Game Over |
